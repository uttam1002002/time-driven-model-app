# How to Decide – A Time-Driven Leadership Tool

*Based on Vroom’s Normative Model of Leadership Decision-Making (Time-Driven)*

A lightweight leadership tool to help leaders decide **how to decide** when
balancing impact, urgency, fairness, and team commitment.

## What it does
- Guides leaders through key situational factors
- Recommends an appropriate decision-making approach
- Helps balance speed, decision quality, and buy-in
